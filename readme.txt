This plugin lets you create Email Subscription Popups using Mailchimp. You can customize the design and configure behavior of the popup. 

== Description ==

This plugin lets you created email subscription popups using Mailchimp. You can customize the design and configure behaviour of the popup on different triggers like on page load, after time delay, when back button is clicked, when a link is clicked or after reading a certain part of article. 
You can view the subscriber's emails in your Mailchimp account.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly
2. Activate **Mailchimp Subscription Popup** through the 'Plugins' screen in WordPress
3. **Mailchimp Subscription Popup** now appears in your dashboard
4. Click on Mailchimp Subscription Popup in your dashboard
5. View the tutorial video and click on 'Got it. Thanks' button
6. Select the Email Popup Theme you want
7. Connect with your Mailchimp account
8. Select your email list, to which new subscribers will be added
9. Choose if you want users name
10. Change components to suit your website and usage
11. Click **Save my design** button at top right to continue
12. Now you will be shown a screen with how your popup is triggered. Click the blue button **Click here to save** to save the popup. 
13. Your popup is now saved!
14. Test the popup by going to any page on your website and moving the mouse towards the close button of the browser
15. The popup will load and on entering name and email, the popup will add the user to your selected subscriber list on Mailchimp, which you can view in your Mailchimp Dashboard.
16. In case it does not work, you can contact us on admin@yeloni.com
